### Discord (EverMind Community)

**Subject: Liminal Space — long-term memory for psychological confrontation**

Hey everyone — sharing what we built with EverMemOS.

**Liminal Space** is a production app that tracks psychological development over time. 7-factor coherence model, 12 Jungian archetype agents, visual artifacts (Imprints) generated from a semiotic compiler. 271 API routes, 125 DB migrations. Not a prototype.

**What EverMemOS gave us:** The memory layer this system was missing. Three channels feed MemCells:

1. Coherence checks (7 psychological factors scored 1-10, tagged by phase/tier)
2. Council deliberations (which archetypes spoke, synthesis, user resonance)
3. Threshold moments (motifs, polarities, shadow elements, coherence snapshot)

**The headline feature — Arc Memory:** Agentic retrieval (`retrieve_method: 'agentic'`) across the full MemScene, presenting a disputable account of your arc. The system confronts: "You've been avoiding connection for nine days. Last time this gap appeared, connection collapsed. You told yourself it was fine. It wasn't." The user pushes back. That correction is the product.

**The correction loop IS the product.** The system makes claims grounded in months of evidence. The user argues. The system learns not from the data—from the argument about what the data means.

**Integration pattern worth sharing:** Liminal has a 15-handler EventBus (analytics, phase updates, XP, etc.) that runs via `Promise.allSettled()`. EverMemOS became the 16th handler — one file, three lines of registration, zero impact on response time. All channels are fire-and-forget with graceful degradation.

Happy to share what we learned about the agentic retrieval API — it's powerful but has some nuances around GET-with-body vs POST for search.
